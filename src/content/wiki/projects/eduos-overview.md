# EduOS Platform

EduOS is a modern educational management system built with Go and React.

## Tech Stack

- **Backend:** Go (Fiber/Chi)
- **Frontend:** React, TanStack Query, TailwindCSS
- **Database:** PostgreSQL

## Current Focus

We are currently working on the student enrollment flow and integrating the TanStack table for the admin dashboard.
