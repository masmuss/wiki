## Root File Conventions

Semua project Wajib memiliki:

<FileTree>

- .gitignore
- README.md
- LICENSE (opsional)
- Makefile (atau just-task, taskfile)

</FileTree>

## Monorepo Structure

Untuk project berskala besar yang terdiri dari multiple packages (sebagai contoh, lihat repositori [masmuss/trak](https://github.com/masmuss/trak)):

<FileTree>

- apps/
  - bot/ <span class="text-muted-foreground ml-2 text-xs font-sans"># Telegram/Discord bot service</span>
  - web/ <span class="text-muted-foreground ml-2 text-xs font-sans"># Web frontend (Next.js/Astro)</span>
- packages/
  - database/ <span class="text-muted-foreground ml-2 text-xs font-sans"># Drizzle schemas, migrations</span>
  - services/ <span class="text-muted-foreground ml-2 text-xs font-sans"># Shared business logic & services</span>
  - shared/ <span class="text-muted-foreground ml-2 text-xs font-sans"># Shared types, constants, utilities</span>
- pnpm-workspace.yaml <span class="text-muted-foreground ml-2 text-xs font-sans"># Workspace config</span>
- turbo.json <span class="text-muted-foreground ml-2 text-xs font-sans"># Turborepo config</span>
- package.json

</FileTree>

## README Wajib Berisi

1. **Judul** dan deskripsi singkat
2. **Prerequisites** (runtime, tools)
3. **Cara install** (`pnpm install`, `go mod download`, dll)
4. **Cara jalanin** (`pnpm dev`, `go run .`, dll)
5. **Testing** (`pnpm test`, `go test ./...`, dll)
6. **Environment variables** (copy `.env.example`)
7. **Database setup** jika ada
8. **Deployment** (jika relevant)
